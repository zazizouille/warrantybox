## GitHub Pages

This repository is configured to serve static content via GitHub Pages. The privacy policy and other documents are accessible through the GitHub Pages URL.

### Available Documents

- [Privacy Policy](privacy-policy.html) - Privacy policy for the WarrantyBox mobile application

## Setup

This site is automatically published via GitHub Pages from the main branch. To make changes:

1. Edit the HTML files in the repository
2. Commit and push your changes
3. GitHub Pages will automatically update the site

## Local Development

To view the site locally, simply open the `index.html` file in your web browser.
